just left or right mouse click on some text (after login). the right mouse click displays the current published text as well.

id attributes

where ever a tag has an id-attribute and an class-attribute with value ixtlan. these tags can be edited via ixtlan.

if there are nested tags then the id-attribute is a |-separated list of ids - each id for each text node the tag contains.

title tag

the title tag carries the default key title. but the value of thatkey can be changed only in the datacentric admin interface

namespace

each page gets its unique namespace for ids, i.e. the ids need to be unique within a page but different pages can have the same ids.
